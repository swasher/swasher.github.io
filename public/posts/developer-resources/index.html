Планирование дел и хранение информации#

  • wunderlist.com - имхо лучший сервис ведения списков (в т.ч. todo)

  • keep.google.com - сервис заметок

  • lastpass - синхронизация паролей браузера

  • keeweb.info - хранилище паролей от “железа” (with dropbox sync)

Облачные хранилища#

  • dropbox.com - синхронизация файлов между копьютерами. Удобно хранить исходники при работе на нескольких девайсах. Free - ~ 22 GB max

  • mega.nz - хранение файлов без синхронизации, с шифрованием на стороне клиента. Free - 50GB

Репозитории исходных кодов#

  • github.com – неограниченное количество бесплатных публичных репозиториев;

  • bitbucket.org – бесплатные приватные репозитории (GIT и Mercurial), с которыми могут работать до пяти человек;

  • Cloud Source Repositories – бесплатные приватные GIT-репозитории, развернутые на облачной платформе Google Cloud. Сервис поддерживает синхронизацию с репозиториями GitHub и Bitbucket. Размер хранилища до 500 Мб.

  • gitkraken - графический клиент для git

IDE и редакторы#

  • pycharm - лучшая IDE для python

  • atom.io - текстовый редактор от GitHub

  • Koding IDE - онлайн IDE, в вирт. машине Ubuntu, плюс групповая работа

  • prose.io - сервис, позволяющий напрямую редактировать файлы в репозитории GitHub

  • c9.io - Cloud9 IDE, мощная онлайн IDE

Онлайн инструменты разработки и тестирования#

  • browserling.com - онлайн браузер

  • browsershots.org - делает скриншот страницы во всех известных браузерах

  • mapbox.com - платформа для разработки приложений на основе карт (гео)

Тестирование кода, css, html, js#

  • repl.it - запуск кода, поддерживает множество языков

  • bootply.com - тестирование html, css, javascript

  • jsfiddle.net - тестирование html, css, javascript

  • codepen.io - тестирование html, css, javascript

  • playcode.io/ - тестирование html, css, javascript

  • codesandbox.io - полноценная IDE

  • bootsnipp.com - множество образцов макетов на бутстрапе, например формы регистрации

  • runnable.io - песочница для запуска приложений, в т.ч python/django

СУБД#

mongotron.io - Cross platform MongoDB management. Open Source, built using Electron and Angular JS

Фронтенд#

  • getbootstrap.com - bootstrap

  • Effeckt.css - css фреймворк от создателей html5 boilerplate. Реализует различные эффекты

  • formvalidation.io - очень крутой фреймворк-валидатор форм, интеграция с bootstrap и другими css фрейворками

  • bootstrap-validator - валидатор форм для bootstrap

  • jqueryvalidation.org - валидатор форм на jquery

  • parsleyjs.org/ - валидатор форм, очень крутой

  • summernote.org - очень простой интегрируемый редактор на бутстрапе

  • textbox.io - интегрируемый редактор, платный, есть фри версия, вроде бы без ограничений

  • bootboxjs.com - js библиотека разнообразных (dialog, modаl, alert, confirm) окон

Шрифты и иконки#

  • fontawesome.io - иконки, есть платная версия fortawesome

  • glyphicons.com - иконки, в бесплатной версии только базовый набор

  • nerdfonts.com - агрегатор иконок

  • programmingfonts.org - большая коллекция IDE-шрифтов в виде фонт-библиотеки

Python#

  • click.pocoo.org - отличный проект для разработки интерфейса командной строки

Утилиты#

  • gifcam - маленькая утилита для снятия гифок с экрана.

Хостинг#

  • netlify.com – хостинг статического сайта или приложения, хранилище 100 Мб, пропускная способность 1 Гб; автоматический деплой при обнаружении изменений в репозитории GitHub

  • cloudinary.com - хостинг изображений. Трансформации изображений. Фри план: 7,500 Monthly Transformations, 75,000 Total Images, 2 GB Managed Storage, 5 GB Monthly Viewing Bandwidth

  • firebase.com - хостинг с бесплатным планом навсегда. В бесплатном плане нет custom domains

  • bitballoon.com - хостинг с бесплатным планом на их субдомене

  • online.net - VPS за три евро

  • ovh.com - VPS 3.5 евро (OpenStack KVM, 1 vCore, 2.4 GHz, 2 GB RAM, 10 GB SSD, Local RAID 10)

  • digitalocean.com - один из самых уважаемых хостеров. VPS за 5 долл (512 MB RAM, 1 CoreProcessor, 20GBSSD Disk, 1TBTransfer)

  • serverbear.com - сводная информация по всем vps

PaaS#

  • heroku - Heroku одно из немногих мест, где действительно бесплатно и качественно можно развернуть django-приложение.

  • paasify.it - подбор Paas по параметрам

Домены#

  • freenom.com - фри домены tk, ml, ga, cf, gq

  • pop.co - фри домен co

  • pp.ua - фри домен pp.ua

Почта#

  • mailjet.com - рассылка писем, множество фич. Free - 6000 emails/month

  • mailgun.com - The Email Service For Developers. Free - 10k emails/month

Auth#

  • auth0.com - Add authentication to your web. Free for 7k users.
